# ELBAH - A Jornada do Amuleto Perdido

Um RPG em console desenvolvido em **C++** para a disciplina da UNAERP. O jogo oferece uma experiência clássica de RPG de texto com sistema de personagens, inventário, combate e perseguição de missões épicas.

## 📋 Características

- **Sistema de Personagens**: Crie personagens com diferentes classes e raças
- **Atributos Dinâmicos**: Força, Inteligência, Destreza, Carisma e outros atributos que influenciam o gameplay
- **Sistema de Inventário**: Colete e gerencie itens com até 50 slots
- **Sistema de Combate**: Enfrente mobs e inimigos em batalhas baseadas em atributos
- **Menu Principal**: Opções de novo jogo, carregar jogo, avisos e créditos
- **Interface Interativa**: Textos animados com efeito de digitação em tempo real

## 🎮 Como Jogar

### Compilação
```bash
g++ RPG.cpp -o RPG.exe
```

### Execução
```bash
RPG.exe
```

## 🎯 Objetivos do Projeto

Encontrar o Amuleto Perdido e completar a jornada épica através de:
- Exploração de diferentes áreas e encontros
- Desenvolvimento de habilidades do personagem
- Coleta e equipamento de itens poderosos
- Confronto com mobs e chefes
